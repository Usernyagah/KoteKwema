# Rotis Fonts

Place your Rotis font files in this directory.

Required font files:
- RotisSansSerif-Regular.woff2
- RotisSansSerif-Regular.woff
- RotisSansSerif-Light.woff2
- RotisSansSerif-Light.woff
- RotisSansSerif-Bold.woff2
- RotisSansSerif-Bold.woff

The fonts are configured in `src/index.css` and will be used throughout the application.
